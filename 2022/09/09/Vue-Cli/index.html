Nodejs

一、认识Nodejs

Node.js® 是一个基于 Chrome V8 引擎 的 JavaScript 运行时。

1.1 宿主环境

Javascript 是由 ECMAScript、DOM、BOM 三部分组成，浏览器实现了 ECMAScript 标准，使其能够运行于浏览器，我们称浏览器为 ECMAScript 的宿主环境。

Nodejs就是另一种可以运行 ECMAScript 的宿主环境。

1.2 安装

Nodejs 开源且跨平台，在 Windows 平台，Nodejs 就是一个以 .msi 结尾的安装程序，分为 32位 和 64位 两个版本，双击运行根据引导程序完成安装即可。

Nodejs中文官网

打开 cmd 或 powershell 执行如下命令，如出现版本则为安装成功。

1.3 体验

Nodejs 程序是以 .js 结尾的文件，创建 demo.js

打开 cmd 或 powershell 命令行工具，执行 demo.js

注：Nodejs 和 JavaScript 都是基于 ECMAScript 语言规范的，二者的语法具有一致性，显著的区别是 Nodejs 中并没有 DOM 和 BOM，因此无法使用 window 或 document 相关语法。

注：创建文件名或者文件夹的名字一定一定一定要避免使用 node。

二、NPM

Nodejs 有着庞大的社区支持，全世界优秀的开发人员封装了大量实用的模块，托管在开源平台上供大家随意下载使用。

为了方便大家上传、下载开源模块，社区还开发专门的工具 npm，该工具随着 Nodejs 一起被安装了。

2.1 NPM基本使用

  1. NPM(Node Package Manager), node的包管理器

  2. 只要安装Node软件包，那么当前电脑中就会自带Npm包管理器

  3. 切记，为下载第三方模块更加快速，一定要将镜像源设置成国内淘宝服务器

2.2 NPM常用命令
命令 功能
npm init [-y] 初始化一个package.json文件
npm install 包名 –save 安装一个包，会生成node_modules文件夹
npm install 包名 -g 安装一个全局命令行工具
npm install 包名 –save 将安装的包添加到package的依赖中(dependencies)
npm uninstall 包 卸载当前目录下某个包
npm uninstall 包 -g 卸载全局路径安装下的某个包
npm list 查看当前目录下安装的所有包
npm list –g 查看全局路径下安装的所有的包
2.3 NRM基本使用

NRM ( Node Regitry Manager )，是Npm的镜像源管理工具

  1. 有时候国外资源太慢，使用这个就可以快速地在npm源间切换

  2. 安装nrm： 在命令行执行命令，npm install nrm -g ，全局安装nrm

  3. 执行命令nrm ls查看可选的源

  4. 可以通过nrm test测试相应源的相应时间

2.4 NRM基本命令
命令 功能
nrm ls 查看镜像源
nrm test 检测镜像源下载速度
nrm use 切换镜像源
nrm current 当前镜像源
nrm -h 查看nrm命令
2.5 描述性文件

若干模块组合起来放在统一目录中时推荐创建 package.json 文件，该文件中记录模块的名称、版本、依赖等信息，package.json 文件即可以手动创建，也能通过命令快速创建。

  1. name 标记模块的名称，名称中不允许包含大写字母

  2. version 标记模块的版本号

  3. main 模块入口，默认为 index.js

  4. script 自定义脚本命令，以 npm run 命令名称 形式调用，npm run start 时，可以省略 run

  5. dependencies 记录所有被依赖的模块，安装模块时添加 --save 或 -S 模块会被记录在 package.json 中

  1. devDependencies 记录开发阶段所有被依赖的模块，安装模块时添加 --save-dev 或 -D

    注：设置淘宝镜像可以加快模块的安装速度 npm config set registry https://registry.npm.taobao.org

组件

一、组件化开发
  1. 组件化开发指的是：根据封装的思想，把页面上可重用的 UI 结构封装为组件，从而方便项目的开发和维护
  2. vue 是一个支持组件化开发的前端框架。 vue 中规定：组件的后缀名是 .vue。之前接触到的 App.vue 文件本质上就是一个 vue 的组件
二、vue 组件三个组成部分

每个 .vue 组件都由 3 部分构成，分别是：

  1. template -> 组件的模板结构
  2. script -> 组件的 JavaScript 行为
  3. style -> 组件的样式 其中，每个组件中必须包含 template 模板结构，而 script 行为和 style 样式是可选的组成部分
